# Maven Java Project

This is a simple Maven Java project created for learning Maven and GitHub integration.

## Technologies Used

* Java
* Maven
* Git
* GitHub

## Project Structure

* `src/main/java` → Main Java source files
* `src/test/java` → Test files
* `pom.xml` → Maven configuration

## Run Project

```bash id="r5n1vk"
mvn clean compile
mvn exec:java
```
